# Regression from Scratch

This repository contains implementations of fundamental regression algorithms built completely from scratch using Python, without relying on machine learning libraries such as scikit-learn.

## Implemented Algorithms

### 1. Linear Regression
- Implemented using Gradient Descent
- Manual computation of gradients
- Mean Squared Error (MSE) loss
- Data visualization using Matplotlib

### 2. Logistic Regression
- Implemented using Gradient Descent
- Sigmoid activation function
- Binary Cross-Entropy Loss
- Binary classification
- Accuracy evaluation
- Sigmoid curve visualization

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib

## Objective

The objective of this repository is to understand the mathematical foundations of regression algorithms by implementing them from scratch instead of using pre-built machine learning libraries.
